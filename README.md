Next.js Login & Dashboard App

A simple web application built with Next.js + TypeScript that demonstrates a basic authentication flow with route protection and a dashboard.

Features

1.Authentication

Login page with username & password
Mock API-based credential validation
    -Login Credentials:
    1. Username-admin
    2. Password-1234
Error handling for invalid login attempts
Session-based authentication

2.Dashboard

Displays logged-in username (top-left)
Profile icon with logout option (top-right)
Protected route (accessible only after login)

3.Logout

Clears session/token
Redirects back to login page

4.Tech Stack

Framework: Next.js
Language: TypeScript
Styling: React Bootstrap
Routing: Next.js App Router 
State Management: Redux Toolkit

Instructions to run the project

1.Enter in project directory
2.Then Install Dependencies by running 'npm i'.
3.Then run 'npm run dev' to start the project.
